gogui

gogui, gogui.jar — Graphical user interface for Go programs

Synopsis

gogui [-analyze command] [-auto] [ -computer-black | -computer-both | -computer-none ] [-command command] [-file filename] [-fillpasses] [-gtpshell] [-gtpfile filename] [-help] [-komi value] [-move n] [-rules chinese | japanese ] [-size n] [-time timespec] [-verbose] [-version] [program]

Description

GoGui is a graphical interface to programs that play the game of Go and use GTP (Go Text Protocol).

Options

-analyze command
Initialize analyze command.
-auto
Automatically start a new game if the program plays both colors and a game is finished.
-command command
Send Gtp command at startup. The argument can contain multiple commands separated by '\n'.
-computer-black
Computer plays black.
-computer-both
Computer plays both sides.
-computer-none
Computer plays no side.
-file filename
Load SGF file.
-fillpasses
Never send subsequent moves of the same color to the program.
-gtpshell
Open GTP shell at startup.
-gtpfile filename
Send GTP file at startup.
-help
Display help and exit.
-komi value
Set the komi value.
-move n
Load SGF file until move number.
-rules chinese | japanese
Set rules.
-size n
Set board size.
-time timespec
Set time limits (min[+min/moves]).
-verbose
Print debugging messages.
-version
Print version and exit.
